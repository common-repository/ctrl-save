== Description ==
You can save the file by pressing ctrl + s.
If it is not published yet, it will be saved as a draft.

If you want to publish it, use ctrl + shift + s to do so.


== Installation ==
1. Upload the plugin files to the `/wp-content/plugins/ctrl+s` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress.
